## !!steps Creating a function

Functions are first-class citizens in Lua. They can be stored in variables, passed as arguments to other functions, and returned as values from other functions.

```lua ! example.lua

print("Hello, World!")

function sum(a, b)
  return a + b
end

local a = 3
local b = 5

print(sum(a, b)) -- Output: 8
```


## !!steps Using tables

Tables are the only data structure in Lua. They can be used as arrays, dictionaries, or even objects.

```lua ! example.lua

function sum(a, b)
  return a + b
end

-- Creating a table
local calculator = {
  version = "1.0",
  add = sum,
}

print(calculator.add(10, 20)) -- Output: 30
```

## !!steps Tables are references

Tables in Lua are reference types. Don't worry about memory management.

```lua ! example.lua

a = {}
a["x"] = 10
b = a
-- `b' refers to the same table as `a'
print(b["x"])  --> 10
b["x"] = 20
print(a["x"])  --> 20
a = nil
-- now only `b' still refers to the table
b = nil
-- now the table is free to be collected
```


## !!steps Using Löve

Love event cycle consists of several callback functions that you can define in your code. The most commonly used callbacks are love.load, love.update, and love.draw.

```lua ! love.lua
function love.load()
  -- called once at the beginning of the game
end

function love.update(dt)
  -- called every frame to update the game state
end

function love.draw()
  -- called every frame to render the game
end
```

## !!steps Drawing text

You can use the love.graphics.print function to draw text on the screen.

```lua ! love.lua
function love.draw()
  love.graphics.print("Hello World!", 400, 300)
end
```

## !!steps Drawing an image

You can use the love.graphics.draw function to draw images on the screen.

```lua ! love.lua
function love.load()
  whale = love.graphics.newImage("whale.png")
end

function love.draw()
  love.graphics.draw(whale, 300, 200)
end
```

## !!steps Moving an object
You can update the position of an object in the love.update function.

```lua ! love.lua
local x = 100
local y = 100
local speed = 200

function love.update(dt)
  if love.keyboard.isDown("right") then
    x = x + speed * dt
  elseif love.keyboard.isDown("left") then
    x = x - speed * dt
  end

  if love.keyboard.isDown("down") then
    y = y + speed * dt
  elseif love.keyboard.isDown("up") then
    y = y - speed * dt
  end
end

function love.draw()
  love.graphics.rectangle("fill", x, y, 50, 50)
end
```

## !!steps Delta time
Delta time (dt) is the time elapsed since the last frame. It is used to make movement frame-rate independent.

```lua ! love.lua
function love.update(dt)
  -- Expensive computation to simulate a heavy load
  for i = 1, 1000 do
    for j = 1, 1000 do
      local y = math.sqrt(i) * math.cos(i)
    end
  end

  print(dt) -- Print delta time to the console
end

-- Frame 1: dt = 0.084927708333333
-- Frame 2: dt = 0.0468515
-- Frame 3: dt = 0.028987291666667
```
