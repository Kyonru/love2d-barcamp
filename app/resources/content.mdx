## !!steps Awesome Löve

A categorized community-driven collection of high-quality, awesome LÖVE libraries, projects, and resources.

[Visit Page](https://github.com/love2d-community/awesome-love2d)

### !content

#### 🎮 Game Frameworks & State Management

- [Baton](https://github.com/tesselode/baton) — Input handling for LÖVE.
- [lua-state-machine](https://github.com/kyleconroy/lua-state-machine) — Finite state machine in Lua.
- [HUMP](https://github.com/vrld/hump) — Utility library (camera, gamestate, vector math) for LÖVE.
- [Shöve](https://github.com/Oval-Tutu/shove) — Scene and UI management for LÖVE.
- [bootstrap-love2d-project](https://github.com/Oval-Tutu/bootstrap-love2d-project) — LÖVE project template with multi-platform build support.

#### 🤖 AI & Behavior

- [beehive.lua](https://github.com/drhayes/beehive.lua) — Behavior tree library for AI in Lua.

#### 💬 Dialogue & Narrative

- [LoveDialogue](https://github.com/Miisan-png/Love-Dialogue) — Dialogue engine with branching and rich text effects.
- [SYSL-Text](https://github.com/sysl-dev/SYSL-Text) — Styled text rendering with tag-based effects.

#### ✨ Effects & Particles

- [HotParticles](https://github.com/ReFreezed/HotParticles) — Visual particle editor for LÖVE.
- [Moonshine](https://github.com/vrld/moonshine) — Shaders and post-processing effects for LÖVE.

#### 🏰 Procedural Generation

- [Astray](https://github.com/SiENcE/astray) — Procedural maze and dungeon generator in Lua.

#### 🎵 Audio Tools

- [lovebpm](https://github.com/rxi/lovebpm) — Sync events to the BPM of an audio track.

#### 🧩 Development Tools

- [Feather](https://github.com/Kyonru/feather) — Plugin-based debugger for LÖVE.
- [Sprite Sheet Helper](https://sprite-sheet-helper.up.railway.app/) — Generate sprite sheets from 3D objects.
